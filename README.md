# ccstudy-33
ビットコインとか勉強会#33の発表資料とサンプルコードです。  
[発表資料](https://haryu703.github.io/ccstudy-33/index.html)

## Usage
- スライドを生成する  
    `$ yarn slide`  
    詳しいオプションは [marp-cli](https://github.com/marp-team/marp-cli#basic-usage) 参照
- P2PKH のサンプルコードを動かす  
    `$ yarn p2pkh`
- Mecenas のサンプルコードを動かす  
    `$ yarn mecenas`
